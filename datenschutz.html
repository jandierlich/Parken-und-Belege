2. Datenverarbeitung durch die App

"Parken und Belege" verarbeitet folgende Daten ausschließlich lokal in deinem Browser auf deinem Gerät: Fotos von Parkscheinen und Nebenkosten-Belegen; manuell oder per Texterkennung erfasste Angaben (Datum, Ort/Titel, Betrag, Kategorie, Notizen); Standortdaten, ausschließlich zur Anzeige des zuletzt erfassten Parkorts auf einer Karte.

3. Keine Übermittlung an unsere Server

Es findet keine Übertragung dieser Daten an uns oder an Dritte statt. Die Texterkennung (OCR) läuft vollständig offline im Browser. Es werden keine Nutzungsdaten, keine Analyse- oder Trackingdienste eingesetzt.

4. Hosting & technische Bibliotheken

Die Web-App wird über GitHub Pages (Microsoft/GitHub) bereitgestellt. Beim Aufruf verarbeitet GitHub technisch notwendige Server-Logs (u. a. IP-Adresse). Zur Funktion werden zudem folgende externe Bibliotheken nachgeladen: Leaflet (unpkg.com), Tesseract.js (jsdelivr.net), jsPDF (cdnjs.cloudflare.com) — dabei wird die IP-Adresse technisch notwendig an die jeweiligen Anbieter übertragen. Es kommen ausschließlich Systemschriften zum Einsatz, keine Google Fonts.

5. Kartendarstellung

Kartendaten: © OpenStreetMap-Mitwirkende, Open Database License.

6. Kamera-, Standort- und Mikrofonzugriff

Die App fragt Zugriff auf Kamera (zum Scannen von Belegen), Standort (zur Kartenanzeige) und Mikrofon (für optionale Sprachnotizen) an. Sprachnotizen werden ebenso wie Fotos ausschließlich lokal auf deinem Gerät gespeichert, nicht übertragen. Diese Berechtigungen können jederzeit in den Safari-/iPhone-Einstellungen widerrufen werden.

7. Speicherdauer und Löschung

Die Daten verbleiben im Browser-Speicher, bis du sie selbst löschst oder die Website-Daten in Safari leerst. Ein manueller Export/Backup ist über die Download-Funktion in der App möglich.
